# AI Agents Assignment - Smart Manufacturing Simulation

## Description
This repository demonstrates a comprehensive AI Agent strategy for Smart Manufacturing at AutoParts Inc. It includes:

- Quality Control Agent for defect detection
- Maintenance Agent for predictive maintenance
- Production Scheduling Agent for optimizing workflow

## Folder Structure

AI-Agents-Assignment/

├── notebooks/

│ └── agent_simulation.ipynb

├── src/

│ ├── agent_definitions.py

│ ├── agent_simulation.py

│ └── utils.py

├── diagrams/

│ └── agent_workflow.txt

├── simulations/

│ └── n8n_simulation_link.txt

├── data/

│ └── sample_production_data.csv

└── README.md


## Tools & Libraries
- Python 3.x
- Matplotlib
- n8n / Make.com for workflow simulation

## Instructions
1. Run Python simulation: `python src/agent_simulation.py`
2. Open Jupyter notebook for visualization: `notebooks/agent_simulation.ipynb`
3. Access simulation workflows: `simulations/n8n_simulation_link.txt`

## Key Benefits
- Reduced defect rates
- Predictive maintenance → less downtime
- Optimized production scheduling → faster delivery & lower labor cost
